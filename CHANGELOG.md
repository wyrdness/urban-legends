# Changelog

All notable changes to this repository will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial repository structure
- Comprehensive README.md with phenomenon documentation
- api.json following Wyrdness schema v1.0.0
- SOURCES.md bibliography
- Issue templates for legend reports, corrections, and source additions
- Pull request template

### Data Included
- Basic phenomenon metadata and classification (URBAN_LEGEND)
- Legend types (Supernatural, Cautionary, Contamination, Conspiracy)
- 11+ famous legends (Bloody Mary, Slender Man, Hookman, Kidney Theft, etc.)
- FOAF narrative structure documentation
- Geographic distribution with regional variants
- Theories (sociological, psychological, anthropological, media studies)
- 2 case studies (Slender Man stabbing, Satanic Panic)
- Historical timeline (1800s-TikTok era)
- Cultural significance and internet memetics
- Jan Harold Brunvand research documentation
